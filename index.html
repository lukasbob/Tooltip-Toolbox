Tooltip-Toolbox by lukasbob

A highly flexible jQuery Tooltip plugin that is easily adaptable to create complex non-modal overlays, such as dropdown menus etc.

Dependencies

jQuery 1.3.2

Authors

Mads Jacobsen (lukasbob@gmail.com)

Contact

Mads Jacobsen (lukasbob@gmail.com)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/lukasbob/Tooltip-Toolbox
